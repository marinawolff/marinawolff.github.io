What is Bootstrap?

Bootstrap is a free open-source CSS framework created to help developers to build quickly and easily responsive, mobile-ready websites that are compatible with many browsers. Bootstrap features a 12-column grid and components ready to use.

Installation

Installation methods:

  • CSS and JS: Download the pre-complied bootstrap CSS and JavaScript files from bootstrap website. This method works well because it will work without an internet connection as well.
  • Bootstrap CDN: Place that hosts libraries like bootstrap. The advantage of using CDN is that the website loads faster if the browser already has the cache version in its memory.
